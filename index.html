Pro Outline By Normal Experimental

Advanced normal-based outline shader for Unity URP

Overview

The Pro Outline By Normal Experimental shader creates high-quality outlines based on normal map differences and depth values. It's designed to produce clean, precise outlines with extensive customization options for artistic control.

Precise Edge Detection

Uses normal cusp detection to create accurate outlines along geometry edges.

Distance-Based Fading

Intelligent depth-based fade with exponential falloff for natural distance attenuation.

High Customization

Extensive parameters for precise artistic control over outline appearance.

Stylization Options

Paper/ink mode for stylized rendering effects.

Parameters

Outline Appearance

Outline Color Color

Sets the color of the outline effect.

Thickness Float

Controls the width of the outline. Higher values create thicker lines.

Edge Detection

Use Angle Edge Threshold Boolean

When enabled, the shader uses angular differences between normals for edge detection.

Normal Threshold Float

Controls edge detection sensitivity using simple normal differences. Used when "Use Angle Edge Threshold" is disabled.

Edge Threshold Float

Determines how sensitive the edge detection is. Lower values detect more subtle edges, higher values only catch prominent edges.

Normal Intensity Float

Multiplier for the strength of normal-based edges. Higher values create more pronounced outlines.

Depth Intensity Float

Multiplier for the strength of depth-based edges. Higher values create more pronounced outlines at object boundaries.

Depth Control

Use Depth Fade Boolean

When enabled, outlines will gradually fade out with distance using the Max/Fade Distance parameters.

Max Distance Float

Maximum distance from the camera where outlines will render. Objects beyond this distance won't show outlines.

Fade Distance Float

Distance at which outlines begin to fade out. Should be less than Max Distance.

Fade Options

Use Exponential Fade Boolean

When enabled, applies an exponential curve to distance fading for more natural-looking falloff.

Fade Power Float

Controls how quickly the exponential fade occurs. Higher values create a more rapid initial falloff.

Stylization

Paper Ink Boolean

When enabled, creates a stylized ink drawing effect.

(Ink Only) - Bit Texture Blend Float

Controls the blending of the ink effect when "Paper Ink" is enabled.

Albedo Color Color

Base color used for the shader. Can be used to tint the underlying surface.

Usage Tips

  • Finding the Right Thickness

    Start with a low Thickness value (around 1.0) and gradually increase until you achieve the desired look. Remember that thicker lines may require adjusting the Edge Threshold to prevent unwanted artifacts.

  • Edge Detection Tuning

    For precise geometric outlines, enable "Use Angle Edge Threshold" and adjust the Edge Threshold. For softer, more stylized looks, disable angle thresholding and use the Normal Threshold instead.

  • Distance Control

    Set Max Distance based on your scene size - typically slightly less than your camera's far clip plane. Set Fade Distance to 50-80% of your Max Distance for smooth transitions.

  • Exponential Fade for Realism

    Enable "Use Exponential Fade" for more natural distance falloff, especially in larger scenes. Higher Fade Power values create a more immediate drop-off of the effect.

  • Performance Considerations

    Higher Thickness values require more samples and may impact performance. Consider using lower Edge Threshold values for distant objects to reduce visual noise.

  • Stylized Looks

    Enable "Paper Ink" for comic book or cel-shaded styles. Combine with Normal Intensity adjustments to emphasize specific geometric features.

Troubleshooting

Issue Solution
Jagged or Flickering Outlines Try increasing the Edge Threshold slightly or reducing the Thickness.
Outlines Too Subtle Increase Normal Intensity or Depth Intensity, or reduce Edge Threshold.
Unwanted Outlines Increase Edge Threshold to ignore smaller normal variations.
Outlines Disappearing Too Quickly Adjust Fade Distance and Max Distance values, or reduce Fade Power if using exponential fade.
Missing Outlines on Some Objects Ensure that DepthNormalPrepass is enabled in your Renderer Feature settings.
Performance Issues Try reducing Thickness, increase Edge Threshold, or limit the effect to closer objects.

Implementation

Setting Up the Renderer Feature

To use this shader in your URP project, you'll need to add the ProArtisticOutlineRenderer as a Renderer Feature in your URP Asset.

// 1. Add the ProArtisticOutlineRenderer.cs script to your project // 2. In your URP Renderer Asset, click "Add Renderer Feature" // 3. Select "Pro Artistic Outline Renderer" // 4. Configure the settings in the inspector

Required URP Setup

For best results, ensure your URP settings include:

  • Depth Texture enabled
  • Opaque Texture enabled
  • Depth Normal Prepass enabled (done automatically by the renderer feature)

Depth-Normal Prepass Importance

The normal-based outline effect requires a depth-normal prepass to work correctly. This ensures all objects contribute to the outline effect, even if they might be skipped in regular rendering optimizations.

The ProArtisticOutlineRenderer automatically configures this requirement, but if you're implementing your own version, make sure to include:

// In your renderer feature public override void SetupRenderPasses(ScriptableRenderer renderer, in RenderingData renderingData) { // Tell URP that this renderer feature needs the depth and normal textures outlinePass.ConfigureInput(ScriptableRenderPassInput.Normal | ScriptableRenderPassInput.Depth); }
